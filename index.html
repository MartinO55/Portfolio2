Seagull!

This was the first game I built and published with the Unity Engine (and it shows), with the goal of gaining some experience with as much of the game development pipeline as possible. I did all of the programming, 3D modelling, rigging and animating myself; in fact the only thing I didn't do was the sound composition! Unfortunately, this was before I discovered Git and github, so there is no history or source code available, but I have uploaded the completed .apk file to github for you to download. It should run on any android device.

Eshop

This is a full stack web application for an online store selling impractical and overpriced clothes for pets. The front end is a Javascript react application built with vite, while the backend is a google firestore storage box that holds all the product information. Styling has been done via SASS.

Morse Code Translator

This website is a morse code translator built in HTML and javascript for the functionality, with SASS used for the styling. The website translates english (Technially Roman letters) and numbers to Morse Code and back again. This website was also built with a test driven development cycle for the javascript portion, with the tests still in the github if you want to see them.

Google Books

This is a front end website built in react that you can use to search the google books API. The styling for the page has been done in SASS with the rest built in React. The application takes a search term in as a state and uses it as a query to the google books API, then returns 10 results and displays them in cards.
